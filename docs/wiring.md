# Wiring
Using [Ultimarc](https://www.ultimarc.com/) trackballs and spinners, this would be the standard wiring.

### Trackball
| Wire | Connection | Function |
|----------|----------|----------|
| Red | A | 5 Volts |
| Black | A | Ground |
| Purple | A | X1 |
| Blue | A | X2 |
| Red | B | 5 Volts |
| Black | B | Ground |
| Yellow | B | Y1 |
| Green | B | Y2 |

## Spinner
| Wire | Function |
|----------|----------|
| Brown | 5 Volts |
| White | Ground |
| Yellow | X1 |
| Green | X2 |
